# session-loam

> Memory-continuity substrate for agentic workflows. The fourth leg next to [session-essence](https://github.com/dpdanpittman/session-essence), [swarm-lib](https://github.com/dpdanpittman/swarm-lib), and [tribunal](https://github.com/dpdanpittman/tribunal).

**Status**: v0.1 — DESIGN DRAFT. No code yet. Read [`DESIGN.md`](DESIGN.md) first.

essence persists *who* an agent is. swarm-lib persists *what* an agent is doing. tribunal verifies *whether* an agent's output is trustworthy.

session-loam persists *what an agent has learned* — across sessions, across compaction, across crashes. Tag + full-text retrieval, identity-scoped, never-delete by default with reinforce-on-retrieve.
